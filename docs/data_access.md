# Data Access and Licensing

Due to licensing restrictions, raw viral sequence data (e.g., Influenza A HA
sequences from GISAID) cannot be redistributed or included in this repository.

## Molecular Data
- Source: GISAID
- Access: Requires registration and agreement with GISAID terms
- Usage: Raw sequences were used locally; only derived features are stored here

## Epidemiological Data
- Source: WHO FluNet
- Data type: Aggregated weekly case counts
- License: Publicly available

## Reproducibility
All data processing steps are documented in the notebooks, allowing authorized
users to reproduce the results by independently accessing the original data.
